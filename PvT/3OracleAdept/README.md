## 戰術執行重點
1. 第一隻先知騷擾以安全為原則，可以沒賺但不能掉		
2. 上斜坡前先用先知技能天啟確認是否有詭雷及防守配置		
3. 若有詭雷視情況用教士引掉或利用鳳凰引導詭雷炸向敵方		
4. 有飛彈車可用先知集火擊殺
