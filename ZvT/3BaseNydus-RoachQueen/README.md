## 戰術執行重點
1. 早期飛兩隻王蟲，使蟲洞位置有更多選擇
2. 後續不停接蟑螂、后蟲
3. 若對手出女妖，則運輸2~3隻工兵到前線蓋孢子爬行蟲
4. 早期須有效控制對手偵查，避免瓦斯、二本或蟲洞被偵查到
