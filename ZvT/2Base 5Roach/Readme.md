## 戰術執行重點
1. 前4隻異化蟲須繞開死神至人類二礦，並盡可能拖延對方2礦進度
2. 前壓的蟑螂視情況變劫毀獸
3. 蟑螂出門後開始補工兵，並從2礦鋪出蟲苔
4. 此戰術僅為前壓，需自行練習判斷進攻時的攻擊順序
