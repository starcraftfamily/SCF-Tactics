## 戰術執行重點
1. 第一隻后蟲完成後噴卵並移至二礦
2. 后蟲整場僅噴卵2次
3. 前2隻王蟲為開蟲動用，須謹慎保存
4. 早期生產的異化蟲儘量不要讓對方看到
5. 開蟲洞時預留一批異化蟲在對方2礦外，做雙線進攻
