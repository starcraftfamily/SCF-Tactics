## 戰術執行重點
1. 蟑螂繁殖場藏於主礦後方，並用早期異化蟲驅趕工兵教士避免被看到
2. 進攻重點為神族水晶塔		
3. 視情況於前線變劫毀獸		
4. 破門後可全接異化蟲以達到接兵最高效率		
5. 前置基地不一定要放神族三礦，儘可能選擇距離近又不容易被發現的位置
