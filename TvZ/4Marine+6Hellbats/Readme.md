## 戰術執行重點
1. 兵工廠儘可能隱藏並用陸戰隊驅趕王蟲避免被偵查到
2. 惡狼在開戰前先變身戰狼再開戰
3. 戰狼應儘可能聚集，避免被個別擊破
4. 此戰術僅為前壓，後續可依前壓成效決定開三礦或補至3兵營
