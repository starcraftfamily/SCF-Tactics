## 戰術執行重點：		
1. 流程時間點須盡可能準確，每秒的誤差都可能造成戰術失敗		
2. 兵線生產不可間斷，亦不可卡人口		
3. 醫療艇完成時載陸戰隊至前線騷擾，可不賺但不可死		
4. 利用醫療艇+飛彈車控圖確保坦克工兵前進路線安全		
5. 每一隻陸戰隊都極為重要，儘可能保存、操作		
6. 快到對方家時需注意前進蓋碉堡的機會，太過躁進容易失敗
